ERROR:
------
❌ Firebase: This domain is not authorized for OAuth operations for your 
Firebase project. Edit the list of authorized domains from the Firebase 
console. (auth/unauthorized-domain).

CAUSE:
------
Firebase doesn't recognize localhost:5000 as an authorized domain.

SOLUTION (5 STEPS):
-------------------

STEP 1: Open Firebase Console
   → Go to: https://console.firebase.google.com/
   → Select project: unvibe-54ae1

STEP 2: Go to Authentication Settings
   → Left sidebar: Click "Authentication"
   → Top menu: Click Settings (⚙️ gear icon)

STEP 3: Find Authorized Domains
   → Scroll down to "Authorized domains" section
   → You should see a list of domains

STEP 4: Add localhost:5000
   → Click "Add domain" button
   → Type: localhost:5000
   → Click "Add"

STEP 5: Wait and Test
   → Wait 5-10 seconds for changes to propagate
   → Clear browser cache (Cmd+Shift+Delete on Mac)
   → Hard refresh (Cmd+Shift+R on Mac)
   → Go to http://localhost:5000/login
   → Try Google Sign-In again
