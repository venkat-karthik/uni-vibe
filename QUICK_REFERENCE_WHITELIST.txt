HOW IT WORKS:
-------------
1. First user signs up with any email
   → Email automatically approved
   → User logged in
   → Redirected to dashboard ✅

2. Second user tries to sign up with different email
   → System checks: Is email approved?
   → NO → Error: "Email not approved" ❌

3. Second user signs up with first user's email
   → System checks: Is email approved?
   → YES → Account created ✅

FIRESTORE COLLECTIONS:
----------------------

  - Stores all approved email addresses
  - Auto-created when user signs up
  - Used to validate future signups

ERROR MESSAGES:
---------------
"Email not approved"
  → Email hasn't been used to sign up yet
  → Ask existing member to approve

"Username or email already exists"
  → Email or username already registered
  → Use different email or username

"Invalid email format"
  → Email doesn't have @ or .
  → Enter valid email (user@example.com)
