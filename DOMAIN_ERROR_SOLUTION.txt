                    FIREBASE DOMAIN ERROR - SOLUTION ✅
================================================================================

ERROR YOU'RE SEEING:
-------------------
❌ Firebase: This domain is not authorized for OAuth operations for your 
Firebase project. Edit the list of authorized domains from the Firebase 
console. (auth/unauthorized-domain).

WHY THIS HAPPENS:
-----------------
Firebase doesn't recognize localhost:5000 as an authorized domain for OAuth.

THE FIX (SIMPLE):
-----------------
Add localhost:5000 to your Firebase project's authorized domains list.

QUICK STEPS:
------------
1. Go to: https://console.firebase.google.com/
2. Select: unvibe-54ae1 project
3. Click: Authentication (left sidebar)
4. Click: Settings (⚙️ icon at top)
5. Scroll to: "Authorized domains"
6. Click: "Add domain" button
7. Type: localhost:5000
8. Click: "Add"
9. Wait: 5-10 seconds
10. Clear: Browser cache (Cmd+Shift+Delete on Mac)
11. Refresh: Hard refresh (Cmd+Shift+R on Mac)
12. Test: Go to http://localhost:5000/login and try Google Sign-In
